Overload é ter varios métodos com nomes igual mas assinaturas diferentes.
realiza a mesma coisa mas de formas diferentes.
overload == sobrecarga de métodos.

° exemplo:
void imprimeExtrato(){
    // Codigo para 15 dias.
}
void imprimeExtrato(int dias){
    //Codigo para "n" dias.
}

° DRY (Don't repeat yourself)
